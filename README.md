# Consolidação de Medalhas Olímpicas

## Dados utilizados

**Dados históricos (1896–2022):**  
Base dos Dados – Histórico das Olimpíadas  
https://basedosdados.org/dataset/62f8cb83-ac37-48be-874b-b94dd92d3e2b

**Dados Paris 2024:**  
Kaggle – Paris 2024 Olympic Summer Games  
https://www.kaggle.com/datasets/piterfm/paris-2024-olympic-summer-games/data

## Descrição

Este trabalho realiza a consolidação e análise do total de medalhas conquistadas por países ao longo da história dos Jogos Olímpicos, utilizando dados históricos (1896–2022) e dados dos Jogos Olímpicos de Paris 2024.

## Objetivo

Processar e integrar os dados para obter o total de medalhas por país, separando os resultados em:

- Jogos Olímpicos de Verão  
- Jogos Olímpicos de Inverno  
- Total geral de medalhas  

## Resultados

- Três tabelas ordenadas pelo total de medalhas (Verão, Inverno e Total Geral)
- Três gráficos com os 50 países mais bem colocados em cada categoria
